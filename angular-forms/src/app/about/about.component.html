Actuarial Model

The solution is an Angular 14 Application built using the Angular CLI.

Details about the app:

  • The app is using Angular Material components for the side navigation bar, tables, input fields, buttons.
  • The sidenav component was generated using schematics.
  • The form in ActuarialModelComponent is using Reactive forms.
  • The form validations are placed in the form definition.
  • The results (date and percentage) are fomatted using Angular pipes.
  • The ActuarialModelService will be used to get the data from the backend. For now it is just mock data.
